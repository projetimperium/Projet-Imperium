Documentation | SerialMonitor¶

class imperium.SerialMonitor(interval=3)¶
Class SerialMonitor:
  • Responsable a lire le port serie pour des confirmations, warnings, data, etc.
  • Cette instance crée un nouveau processus pour ne pas intervenir avec le serveur web
Parameters:interval (int) – Interval de temps a lire de port serie
onReceiveLine(rawLine)¶

Une ligne de commande a été reçu sur le port serie. La ligne est ensuite analyser selon le format: <PI|module|commande|param

Parameters:rawLine (str) – Ligne de commande
run()¶

Début du Serial Monitor

Lecture du port serial a un interval déterminer. Si une commande est reçu, la fonction onReceiveLine() est appelé
